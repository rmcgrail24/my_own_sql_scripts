Analytic Workspace Manager for Oracle OLAP 11.1.0.7.0B

Installation Notes

Analytic Workspace Manager Version 11.1.0.7.0B

System Requirements

This installation contains the client-side files required to run Analytic Workspace Manager 11.1.0.7.0B. This release is intended for Windows Vista, XP, 2000, Server 2003, and Linux.  It requires at a minimum a connection to Oracle OLAP 11.1.0.6.0 production software or later. In addition, Oracle Database must be in 11.1 compatibility mode or later. To take advantage of all the new capabilities and bug fixes found in Analytic Workspace Manager 11.1.0.7.0B, the Oracle Database 11g Release 11.1.0.7.0 Server Patch must be installed as well.

Major New Features

Analytic Workspace Manager in Release 11.1.0.7.0 provides the ability to add multiple languages to a single analytic workspace.  In addition, individual aggregation definitions may now be defined for each measure of a cube.  The Create Dimension user interface has been modified to allow levels of the dimension to be created at the same time as the dimension.  The functionality of dimension and cube mapping has been enhanced to allow the application to refresh the definitions of database objects interactively to reflect the current state of database schema tables.  Please refer to the Oracle OLAP User’s Guide 11g Release 1 (B28124-03) for further details.

How To Install

Windows and Non-Windows Environments Install Options:

If you have an existing 11.1.0.6.0 database client installation that you wish to update with the AwM 11.1.0.7.0B release, extract the following files from the awm111070B_DBClient.zip file:

awm.jar

awmhelp.jar

xsjwork.jar

awmdep.zip

olap_api.jar

olap_api_spl.jar

awxml.jar

olap_awxml_doc.jar

olap_api_doc.jar

 

Copy the downloaded files as follows:

1.      Unzip the downloaded file into a new directory

2.      Copy awm.jar to [ORACLE_HOME]/olap/jlib

3.      Copy awmhelp.jar to [ORACLE_HOME]/olap/jlib

4.      Copy xsjwork.jar to [ORACLE_HOME]/olap/jlib

5.      Copy awmdep.zip to [ORACLE_HOME]/olap/jlib

6.      Copy olap_api.jar to [ORACLE_HOME]/olap/api/lib

7.      Copy olap_api_spl.jar to [ORACLE_HOME]/olap/api/lib

8.      Copy awxml.jar to [ORACLE_HOME]/olap/api/lib

9.      Copy olap_awxml_doc.jar  to [ORACLE_HOME]/olap/api/doc

10.  Copy olap_api_doc.jar  to [ORACLE_HOME]/olap/api/doc

 

If you wish to install the AwM 11.1.0.7.0B release as a standalone executable jar file, extract the following file from the awm111070B_Standalone.zip file:  awm11.1.0.7.0B.jar

1.      Copy the downloaded file, awm11.1.0.7.0B.jar into a new directory.

2.      Make sure your environment has JRE version 1.5 (Java Runtime Environment) installed.  This is required to be able to run AwM as a single jar file.

3.      Start the awm11.1.0.7.0B.jar file by executing the jar file or by running the jar file from a command line as follows:

java -mx512m -jar awm11.1.0.7.0B.jar
